## 安装和项目初始化

本项目基于Hardhat生成基础项目框架并通过`openzepplin`进行代码编写。

```bash
# install hardhat
npm install --save-dev hardhat
# install openzeppelin/contracts
npm install --save-dev @openzeppelin/contracts
```

```bash
# 初始化
npx hardhat
# 选择 
Create an advanced sample project
```
